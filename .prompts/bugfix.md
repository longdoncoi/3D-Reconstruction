# Bugfix Prompt

1. Reproduce or identify the bug.
2. Analyze architecture and find root cause.
3. Fix bug following SOLID principles.
4. Ensure no MainWindow business logic is added.
5. Build project and run local tests.
6. Check Python linting (if applicable).
7. Run local agent pipeline (agent_pipeline.ps1/sh).
8. Review checklist.
9. Create commit summary.
