# Refactoring Prompt

1. Identify code smells or architectural violations.
2. Create refactoring plan.
3. Ensure no behavior changes.
4. Apply SOLID and Dependency Inversion.
5. Build project and run tests.
6. Check Python linting (Ruff).
7. Run local agent pipeline (agent_pipeline.ps1/sh).
8. Review checklist.
9. Create commit summary.
